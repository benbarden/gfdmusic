# Track catalogue

Working inventory of finished and in-progress tracks across all artist projects.

---

## Workflow

1. **WIP** - In production
2. **Soundcloud Test** - Uploaded to Soundcloud (private or public), using Amplify for 7-day promotion
3. **Ready** - Performed well / mastered / ready for distribution
4. **Released** - Live on streaming platforms

Soundcloud Pro serves as pre-release test bed: mastering, audience testing via Amplify. Tracks that break 1000+ plays are strong release candidates.

---

## Platform notes

### Youtube Music and sample claims

Tracks using vocal samples (especially from Splice) are vulnerable to false copyright claims on Youtube Music. Other producers using the same samples claim ownership, and platforms side with claimants by default.

**Affected releases:**
- GFD 5th album "Anthems" - taken down from Youtube Music
- "Alive" (lead single from Anthems) - initially accepted, later claimed

GFD 6th album used fewer vocal samples as a result but wasn't submitted to Youtube Music.

**Implications for new work:**
- CotU releases should minimize recognizable vocal samples to avoid claim issues
- Instrumental versions of affected tracks could enable Youtube Music presence (e.g., "Alive" instrumental - not yet done)
